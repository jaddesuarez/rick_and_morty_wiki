A flexible input component that supports various types and states.
Built with accessibility and customization in mind.

## Usage

```tsx
import { Input } from "@components/shadcn/Input/Input";

// Basic input
<Input type="text" placeholder="Enter text..." />

// With label
<div className="grid gap-1.5">
  <label htmlFor="email">Email</label>
  <Input type="email" id="email" placeholder="Email" />
</div>
```

## Features

- Multiple input types support
- Disabled state
- Focus and hover states
- Error state support
- Placeholder text
- Custom styling options
- Form integration

## Accessibility

- Proper ARIA attributes
- Label association
- Focus management
- Keyboard navigation
- High contrast states
- Screen reader support

## Best Practices

- Always provide labels for inputs
- Use appropriate input types
- Include placeholder text when helpful
- Handle error states gracefully
- Consider mobile input methods
- Maintain consistent styling
- Follow form validation patterns

The Input component can be used with:

- Helper text
- Icons
- Form validation
