A versatile button component that supports various styles, sizes, and states.
Built with accessibility and customization in mind.

## Usage

```tsx
import { Button } from "@components/shadcn/Button/Button";

// Primary button
<Button>Click me</Button>

// Secondary button
<Button variant="secondary">Settings</Button>

// With icon
<Button>
  <PlusIcon className="mr-2 h-4 w-4" /> Add new
</Button>
```

## Features

- Multiple variants (primary, secondary, destructive, outline, ghost)
- Different sizes (small, default, large)
- Loading state
- Disabled state
- Icon support
- Full width option
- Keyboard navigation

## Accessibility

- Proper ARIA attributes
- Keyboard navigation support
- Focus visible states
- Disabled state handling
- Loading state announcement
- High contrast support

## Best Practices

- Use clear, action-oriented text
- Choose appropriate variants for actions
- Maintain consistent sizing within context
- Include loading states for async actions
- Provide feedback for interactions
- Consider mobile touch targets

## Composition

The Button component can be composed with:

- Icons
- Loading spinners
- Text
- Custom elements via asChild prop
