Toaster is a component that displays a toast message. It is a wrapper around the <a href="https://sonner.emilkowal.ski/" target="_blank">Sonner</a> library.

## Usage

First import it in your layout file:

```tsx
import { Toaster } from "@components/shadcn/Toaster/Toaster";

<Toaster />;
```

Then you can use it in your components:

```tsx
import { toast } from "sonner";

toast.success("Hello");
toast("Hello");
toast.error("Hello");
toast.warning("Hello");
toast.info("Hello");
toast("Hello", {
  description: "Very long description",
  action: <Button>Click me</Button>,
  closeButton: true,
  duration: 1000,
  position: "top-right",
  theme: "light",
  icon: "ðŸš€",
  style: {
    background: "red",
  },
});
```

## Variants

The Toaster component supports the following "variants":

- `default`
- `success`
- `error`
- `warning`
- `info`

## Features

- Customizable through Tailwind CSS
- Supports multiple toasts
- Supports custom icons
- Supports custom actions
- Supports custom styles
