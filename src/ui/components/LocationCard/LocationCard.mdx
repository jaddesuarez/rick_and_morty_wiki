The LocationCard component displays information about a Rick and Morty location.
It features a planet icon, location name, and dimension information in a visually appealing card format.

## Usage

```tsx
import { LocationCard } from "@components/LocationCard/LocationCard.component";

<LocationCard
  id={1}
  name="Earth (C-137)"
  type="Planet"
  dimension="Dimension C-137"
  residents={[]}
  url="https://rickandmortyapi.com/api/location/1"
  created="2017-11-10T12:42:04.162Z"
/>;
```
