The EpisodeCard component displays information about a Rick and Morty episode.
It shows the episode's name, code (e.g., S01E01), and provides navigation to episode details.

## Usage

```tsx
import { EpisodeCard } from "@components/EpisodeCard/EpisodeCard.component";

<EpisodeCard
  id={1}
  name="Pilot"
  episode="S01E01"
  air_date="December 2, 2013"
  characters={[]}
  url="https://rickandmortyapi.com/api/episode/1"
  created="2017-11-10T12:56:33.798Z"
/>;
```

- Displays episode name with animated scroll for long titles
- Shows episode code (season and episode number)
- Favorite episode functionality
- Navigation to episode details
- Responsive design
- Smooth hover animations

- Use consistent episode naming conventions
- Ensure episode codes follow the SxxExx format
- Handle long episode titles gracefully
- Consider mobile viewport sizes
