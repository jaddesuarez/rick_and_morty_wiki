# Page Layout

The PageLayout component is a wrapper that provides a consistent layout for all component documentation pages. It includes a banner at the top and a gap of 100px between the banner and the component examples.

## Usage

```tsx
import PageLayout from "@/components/docs/PageLayout/PageLayout";

<PageLayout>
  <Banner title="Component Name" />
  <Component />
</PageLayout>;
```
