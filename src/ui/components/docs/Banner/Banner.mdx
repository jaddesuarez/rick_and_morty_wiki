# Documentation Banner

A simple banner component used across component documentation pages to display the component name consistently.

## Usage

```tsx
import Banner from "@/components/docs/Banner/Banner";

<Banner title="Component Name" />;
```

## Styling

The banner uses Tailwind CSS classes and follows the application's theme tokens:

- Includes proper spacing and rounded corners
- Fully responsive
